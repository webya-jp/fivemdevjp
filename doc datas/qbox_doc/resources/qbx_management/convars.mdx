# Convars

Below is a list of convars used within qbx_management.
The value shown in the example is the default value for each convar.

## enableGroupManagement

Set to `"true"` to allow bosses to manage job/gang grades. Specifically the grade name.

```lua
setr qbx:enableGroupManagement "false"
```

Type: `boolean`
