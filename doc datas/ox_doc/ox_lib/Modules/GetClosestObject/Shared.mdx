# Shared

## lib.getClosestObject

Get the object handle and coords of the closest object to a set of coordinates.

```lua
lib.getClosestObject(coords, maxDistance)
```

- coords: `vector3`
  - The coords to check from.
- maxDistance?: `number`
  - The max distance to check.
  - Default: `2.0`

Return:

- object?: `number`
- coords?: `vector3`
