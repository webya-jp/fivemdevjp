# Shared

## lib.getClosestPed

Get the ped handle and coords of the closest ped to a set of coordinates.

```lua
lib.getClosestPed(coords, maxDistance)
```

- coords: `vector3`
  - The coords to check from.
- maxDistance?: `number`
  - The max distance to check.
  - Default: `2.0`

Return:

- ped?: `number`
- coords?: `vector3`
