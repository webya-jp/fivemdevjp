# サーバー輸出

## 登録ボスメニュー

ボスメニューを作成する。

::警告
これは実行時のみの変更である。
これは再起動しても持続せず、ファイルも変更しません。
:::

```lua
exports.qbx_management:RegisterBossMenu(menuInfo)
```

* menuInfo：`table`
  * グループ名：`string`
  * タイプだ：`'gang' | 'job'`
  * コーデ：`vector3`
  * サイズは？`vector3`
    * デフォルト：`vec3(1.5, 1.5, 1.5)`
  * 回転？`number`
    * デフォルト：`0.0`
