# コンバース

以下はqbx_management内で使用されるconvarのリストである。
例に示されている値は、各convarのデフォルト値である。

## enableグループ管理

に設定する。`"true"`ボスがジョブ/ギャングのグレードを管理できるようにする。具体的には等級名。

```lua
setr qbx:enableGroupManagement "false"
```

タイプ`boolean`
