# サーバーイベント

::警告
これらのイベント**禁止**他のスクリプトによってトリガーされる。
:::

## ネットワーク外のイベント

### qbx\_diving:server:coralTakenさん

プレイヤーが海底からサンゴを集めたときに発動する。

```lua
AddEventHandler('qbx_diving:server:coralTaken', function(coords) end)
```

* コーデ：`vector3`

## ネットワークイベント

### qbx\_diving:server:sellコーラル

プレーヤーがサンゴを売ろうとすると発動する。\
プレイヤーのインベントリにサンゴがあることを保証するものではない。

```lua
RegisterNetEvent('qbx_diving:server:sellCoral', function() end)
```
