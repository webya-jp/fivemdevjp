# サーバー輸出

## ハスキー

プレイヤーが指定された車両エンティティのキーを持っているかどうかを返します。

```lua
exports.qbx_vehiclekeys:HasKeys(source, vehicle)
```

* ソース`number`
* 車である：`number`

## ギブ・キー

キーをプレイヤーに渡し、ドアのロック解除/ロック解除とエンジンの切り替えができるようにする。
skipNotificationがtrueでない限り、プレイヤーに通知します。

```lua
exports.qbx_vehiclekeys:GiveKeys(source, vehicle, skipNotification)
```

* ソース`number`
* 車である：`number`
* skipNotification?`boolean`

## キーを削除する

プレーヤーからキーを削除します。skipNotificationがtrueでない限り、プレーヤーに通知します。

```lua
exports.qbx_vehiclekeys:RemoveKeys(source, vehicle, skipNotification)
```

* ソース`number`
* 車である：`number`
* skipNotification?`boolean`
