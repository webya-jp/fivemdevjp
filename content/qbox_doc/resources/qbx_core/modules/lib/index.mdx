# リブ

を提供する。`qbx`様々なユーティリティ機能を持つグローバル・テーブル。

## 前提条件

- Lua 5.4 - 追加`lua54 'yes'`をfxmanifestに追加する。
- [オックスリブ](https://coxdocs.dev/ox_lib)- 追加`@ox_lib/init.lua`への`shared_scripts`をfxmanifestに追加してください。

## インストール

追加`@qbx_core/modules/lib.lua`への`shared_scripts`をfxmanifestに追加してください。の下にあることを確認してください。`@ox_lib/init.lua`スクリプト
