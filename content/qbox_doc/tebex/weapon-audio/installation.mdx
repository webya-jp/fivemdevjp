# インストール

## リソースのダウンロード

エスクローの有無にかかわらず、すべてのリソースはCfx.reアセットシステムを使用して配信されます。アセットマネージャーのハブは次のリンクから見つけることができます: https://portal.cfx.re/assets/granted-assets

## FiveMでスクリプトを開始する

1. スクリプトは、あなたの`server.cfg`というフォルダを作成したり、何らかのサブディレクトリを作成したりします。というフォルダを持つことになるでしょう。`resources`ここに、現在使用しているすべてのスクリプトを置きます。ここに`chat_weaponaudio`ここにもある。

2. 以下のコードブロックを`server.cfg`その他`.cfg`サーバ起動時に読み込まれる：

```lua
ensure chat_weaponaudio
```

## weapons.metaファイルの編集

あとは、あなたの`weapons.meta`ファイルを置き換える。`<Audio>`エントリーには`chat_weaponaudio`.これは次のようになる：

### オリジナル

```xml
<Name>WEAPON_M338</Name>
<Model>w_mg_m338</Model>
<Audio>AUDIO_ITEM_COMBATMG</Audio>
<Slot>SLOT_WEAPON_M338</Slot>
```

### 変形

```xml
<Name>WEAPON_M338</Name>
<Model>w_mg_m338</Model>
<Audio>audio_item_m60</Audio>
<Slot>SLOT_WEAPON_M338</Slot>
```

:::info
これは、シグ・ザウエルM338用です。[gta5-mods.com](https://www.gta5-mods.com/weapons/sig-sauer-m338-replace-fivem-animated).これはMGなので、オーディオパックの2つのMGオーディオアイテムのうちの1つを使用しました。
:::

::危険
間違った武器種に間違ったタイプのオーディオアイテムを使わないでください。グロック17はシングルファイア用に作られています。オートマチック武器にこれを適用すると、オーディオがスカスカになります。
:::
