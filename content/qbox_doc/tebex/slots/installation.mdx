# インストール

## リソースのダウンロード

エスクローの有無にかかわらず、すべてのリソースはCfx.reアセットシステムを使用して配信されます。アセットマネージャーのハブは次のリンクから見つけることができます: https://portal.cfx.re/assets/granted-assets

## FiveMでスクリプトを開始する

1. スクリプトは、あなたの`server.cfg`というフォルダを作成したり、何らかのサブディレクトリを作成したりします。というフォルダを持つことになるでしょう。`resources`ここに、現在使用しているすべてのスクリプトを置きます。ここに`qbx_casino_slots`ここにもある。

2. 以下のコードブロックを`server.cfg`その他`.cfg`サーバ起動時に読み込まれる：

```lua
ensure qbx_casino_slots
```

それだけだ！このスクリプトは完全にプラグアンドプレイです。世界中のあらゆるスロットマシンが自動的に動作します。
