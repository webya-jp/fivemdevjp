# はじめに

私たちを応援してくれた瞬間[コ・フィ](https://ko-fi.com/qbox)Discordで適切な役割を受け取ることができます（DiscordアカウントとKo-fiアカウントをリンクして、実際にDiscordの特典を受け取れるようにしてください）。

https://cdn.qbox.re/ にアクセスし、Discordアカウントで認証してください。ここからすべてのダッシュボードにアクセスできるようになります。

- メインダッシュボードには、現在のストレージ使用量、アップロードしたファイルの量、APIトークン（生成された場合）が表示されます。
- 次に、ファイル管理セクションです。ここでは、新しいファイルをアップロードしたり、既存のファイルを削除したり、ファイルの種類をフィルタリングしてリストまたはグリッドビューでファイルを表示したりすることができます。
- そして最後に、組織管理セクションがある。ここでは、新しい組織を作成し、既存の組織を表示し、組織のユーザーを管理することができます。
