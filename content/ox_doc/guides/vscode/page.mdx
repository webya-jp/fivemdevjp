# VSコード

Visual Studio Codeは、Luaを扱う際の推奨エディターです。

## 推奨エクステンション

拡張機能により、VS Codeに言語、デバッガ、その他のツールを追加でき、開発者のエクスペリエンスが向上します。

### sumneko.lua

Lua言語サーバーは、開発をより簡単かつ高速にするための多くの言語機能を提供します。

- 注釈
- オートコンプリート
- コードフォーマット
- 診断／警告
- 動的な型チェック
- 構文チェック

### communityox.cfxlua-vscode-cox

Lua言語サーバーと連動し、Cfxの関数や型宣言を提供。[Luaランタイム](https://github.com/citizenfx/fivem/blob/master/code/components/citizen-scripting-lua/src/LuaScriptRuntime.cpp),[ルア方言](https://github.com/citizenfx/lua/tree/luaglm-dev/cfx),[環境グローバル](https://github.com/citizenfx/fivem/blob/master/data/shared/citizen/scripting/lua/scheduler.lua)そしてゲームネイティブだ。
