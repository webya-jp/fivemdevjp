# ギット

Gitはバージョン管理システムで、変更を効率的に追跡し、マージし、差し戻すことができる。

私たちのリソースはGitHubでホストされているため、変更を追跡し、最新の状態に保つために独自のリポジトリを作成する方法を学ぶことは非常に便利です。もしあなたが私たちのリソースに独自の変更を加える予定があるのなら、これによってあなた自身の変更を保持し、データの損失を防ぐことができます。

* 参照[GitHubのGit設定ガイド](https://docs.github.com/en/get-started/quickstart/set-up-git#setting-up-git).
* のようなGit GUIクライアントを使うことができます。[ギットクラーケン](https://www.gitkraken.com/)ワークフローを改善し、変更をより簡単に維持することができます。
