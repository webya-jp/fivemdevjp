## Ox.CreateAccount

指定された charId または stateId を持つキャラクターの新しいアカウントを作成します。

```lua
Ox.CreateAccount(ownerId, label)
```

**パラメータ**

* ownerId：`number`|`string`
* のラベルが貼られている：`string`

**リターン**

* [`OxAccount`](/ox_core/Classes/Server/OxAccount)

## Ox.CreateGroup

与えられたデータから新しいグループを作成します。

```lua
Ox.CreateGroup(data)
```

**パラメータ**

* のデータがある：`object`
  * と名付けた：`string`
  * のラベルが貼られている：`string`
  * の成績である：`object[]`
    * のラベルが貼られている：`string`
    * accountRole?`string`
  * タイプ？`string`
  * 色？`number`
  * hasAccount?`boolean`

## Ox.SetGroupPermission

指定されたグループと最低等級に権限を割り当て、その権限は上位等級に継承されます。

```lua
Ox.SetGroupPermission(groupName, grade, permission, value)
```

**パラメータ**

* グループ名：`string`
* のグレードである：`number`
* を許可した：`string`
* 値である：`allow`|`deny`

**例**

```lua
-- Adding a permission to a group (server side)
Ox.SetGroupPermission('police', 1, 'handcuff', 'allow')

-- Checking if a player has the 'handcuff' permission
if player.hasPermission('group.police.handcuff') then
    -- Do something
end
```
