# 進捗状況

## lib.progressBar

実行中のプログレスバーを表示します。

<Tabs items={["Lua", "JS"]}>
  <Tabs.Tab>
    ```lua
    lib.progressBar(data)
    ```
  </Tabs.Tab>

  <Tabs.Tab>
    ```ts
    import lib from '@communityox/ox_lib/client'

    lib.progressBar(data)
    ```

  </Tabs.Tab>
</Tabs>

- 期間である：`number`
- のラベルが貼られている：`string`
- useWhileDead?`boolean`
- allowRagdoll?`boolean`
- allowSwimming?`boolean`
- allowCuffed?`boolean`
- allowFalling?`boolean`
- canCancel?`boolean`
- アニメ？`table`(`object`)
  - ディク？`string`
    - シナリオまたはディクテーションのいずれかを指定しなければならない
  - クリップ：`string`
  - フラッグ？`number`
    - デフォルト：`49`
  - ブレンドイン？`float`
    - デフォルト：`3.0`
  - ブレンドアウト？`float`
    - デフォルト：`1.0`
  - 期間？`number`
    - デフォルト：`-1`
  - playbackRate?`number`
    - デフォルト：`0`
  - ロック X？`boolean`
  - ロック Y？`boolean`
  - lockZ？`boolean`
  - シナリオ？`string`
    - シナリオまたはディクテーションのいずれかを指定しなければならない
  - playEnter?`boolean`
    - デフォルト：`true`
- プロップ？`table`(`object`または`array`)
  - \[複数のプロップを定義したい場合は、個別のテーブル（オブジェクトの配列）として渡すことができます。］
  - モデルである：`hash`
  - 骨？`number`
    - デフォルト：`60309`
  - ポーズ：`table`
    - x:`number`
    - y:`number`
    - z:`number`
  - 腐っている：`table`(`object`)
    - x:`number`
    - y:`number`
    - z:`number`
  - rotOrder?`number`
    - [ヨー、ピッチ、ロールの順番。](https://docs.fivem.net/natives/?_0xAFBD61CC738D9EB9)
    - デフォルト：`0`
- 無効にする？`table`(`object`)
  - 移動する？`boolean`
  - 車？`boolean`
  - コンバット？`boolean`
  - マウス？`boolean`
  - スプリント？`boolean`

### 使用例

<Tabs items={["Lua", "JS"]}>
  <Tabs.Tab>
    ```lua
    if lib.progressBar({
        duration = 2000,
        label = 'Drinking water',
        useWhileDead = false,
        canCancel = true,
        disable = {
            car = true,
        },
        anim = {
            dict = 'mp_player_intdrink',
            clip = 'loop_bottle'
        },
        prop = {
            model = `prop_ld_flow_bottle`,
            pos = vec3(0.03, 0.03, 0.02),
            rot = vec3(0.0, 0.0, -1.5)
        },
    }) then print('Do stuff when complete') else print('Do stuff when cancelled') end
    ```
  </Tabs.Tab>

  <Tabs.Tab>
    <Callout>
      この機能は**非同期**を行う必要がある。`.then`プロミスにコールバックをかけるか、あるいは関数を`async`.
    </Callout>

    ```ts
    import lib from '@communityox/ox_lib/client'

    if (await lib.progressBar({
      duration: 2000,
      label: 'Drinking water',
      useWhileDead: false,
      canCancel: true,
      disable: {
        car: true,
      },
      anim: {
        dict: 'mp_player_intdrink',
        clip: 'loop_bottle'
      },
      prop: {
        model: `prop_ld_flow_bottle`,
        pos: {x: 0.03, y: 0.03, z: 0.02},
        rot: {x: 0.0, y: 0.0, z: -1.5}
      },
    })) console.log('Do stuff when complete');
    else console.log('Do stuff when cancelled')
    ```

  </Tabs.Tab>
</Tabs>
