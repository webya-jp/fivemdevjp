# サーバー

組み込み ACL システムのラッパー。lib.addCommand と ox_groups パーミッションを扱います。\
参照[ベーシック・エース＆プリンシプルの概要／ガイド](https://forum.cfx.re/t/basic-aces-principals-overview-guide/90917)をご覧ください。

## lib.addAce

プリンシパルにエースパーミッションを割り当てる。3 番目のパラメータのデフォルトは 'allow' です。`false`はパーミッションを'deny'に設定する。

<Tabs items={["Lua", "JS"]}>
  <Tabs.Tab>
    ```lua
    lib.addAce(principal, ace, allow)

    lib.addAce('group.admin', 'command.say')
    ```

  </Tabs.Tab>

  <Tabs.Tab>
    ```ts
    import lib from '@communityox/ox_lib/server'

    lib.addAce(principal, ace, allow)

    lib.addAce('group.admin', 'command.say')
    ```

  </Tabs.Tab>
</Tabs>

- 校長：`string`
- エース`string`
- を許可する：`boolean`

## lib.removeAce

プリンシパルからエースパーミッションを削除する。3 番目のパラメータのデフォルトは 'allow' です。`false`はパーミッションを'deny'に設定する。

<Tabs items={["Lua", "JS"]}>
  <Tabs.Tab>
    ```lua
    lib.removeAce(principal, ace, allow)

    lib.removeAce('group.admin', 'command.say')
    ```

  </Tabs.Tab>

  <Tabs.Tab>
    ```ts
    import lib from '@communityox/ox_lib/server'

    lib.removeAce(principal, ace, allow)

    lib.removeAce('group.admin', 'command.say')
    ```

  </Tabs.Tab>
</Tabs>

- 校長：`string`
- エース`string`
- を許可する：`boolean`

## lib.addPrincipal

プリンシパルを親プリンシパルに割り当てます。子プリンシパルは親プリンシパルからパーミッションを継承します。

<Tabs items={["Lua", "JS"]}>
  <Tabs.Tab>
    ```lua
    lib.addPrincipal(child, parent)

    lib.addPrincipal('player.1', 'group.moderator')
    ```

  </Tabs.Tab>

  <Tabs.Tab>
    ```ts
    import lib from '@communityox/ox_lib/server'

    lib.addPrincipal(child, parent)

    lib.addPrincipal('player.1', 'group.moderator')
    ```

  </Tabs.Tab>
</Tabs>

- 子供`string`
- 親だ：`string`
