## lib.setVehicleProperties

車両のプロパティ（改造、プレートテキストなど）を設定し、クライアントがそのエンティティを所有している場合は true を返す。

```lua
lib.setVehicleProperties(vehicle, props)
```

- 車である：`entity`
- プロップス`table`

<Tabs items={["Lua", "JS"]}>
  <Tags.Tab>
    ```lua
    RegisterNetEvent('ox_lib:setVehicleProperties', function(netid, data)
        lib.setVehicleProperties(NetToVeh(netid), data)
    end)
    ```
  </Tags.Tab>

  <Tags.Tab>
    ```ts
    import { setVehicleProperties } from '@communityox/ox_lib/client'

    onNet('ox_lib:setVehicleProperties', (netid, data) => {
      setVehicleProperties(NetToVeh(netid), data)
    })
    ```

  </Tags.Tab>
</Tabs>

**リターンズ**

- isEntityOwner：`boolean`

### 推奨使用法

サーバーは、以下のトリガーを使って、プロパティを設定するようにエンティティのオーナーに伝える必要がある。

```lua
TriggerClientEvent('ox_lib:setVehicleProperties', entityOwner, vehNetId, data)
```
