# インターフェース

サーバーのテーマに合わせてUIの原色を変更したい場合は、convarsから簡単に変更できます。

UIを構築する必要はなく、リソースをリスタートさせるだけだ。

コンバース

```yaml
setr ox:primaryColor blue
setr ox:primaryShade 8
```

プライマリーカラーを変更すると、プログレスバー/サークル、スキルチェック、ラジアルメニューセンターボタン、ホバーなどの要素の色が変わります、
ダイアログの確定ボタン、入力フィールドのフォーカスなどです。

プリセットカラーとシェードの全リストはこちらでご覧いただけます：

https://v6.mantine.dev/theming/colors/#default-colors

独自のパレットを作りたい場合は、そのページのガイドに従うことをお勧めする。\
そうすることを念頭に置いてください。*意志*ソースコードをダウンロードし、UIをビルドする必要がある。
