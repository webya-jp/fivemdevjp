# 共有

## lib.string

Lua標準の文字列テーブルを拡張する。

```
string = lib.string
```

### 文字列.ランダム

与えられたパターンに基づいてランダムな文字列を出力する。

```lua
string.random(pattern, length)
```

* パターンがある：`string`
* `1`は0〜9の乱数を出力する。
* `A`はAからZまでのランダムな文字を出力する。
* `a`はa〜zのランダムな文字を出力する。
* `.`はランダムな文字か数字を出力する。
* `^`は文字通り次の文字を出力する。
* それ以外の文字は、その文字を出力する。
* 長さ？`number`
  * 返される文字列の長さを設定し、パディングするか文字を省略する。

戻る：

* 文字列：`string`
