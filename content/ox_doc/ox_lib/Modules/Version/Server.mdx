# サーバー

## lib.versionCheck

リソースのバージョンを GitHub で公開されている最新のリリースと比較します。\
[GitHub のリリース API を利用](https://docs.github.com/en/rest/reference/releases)

```lua
lib.versionCheck(repository)
```

- リポジトリ`string`
