# 牛燃料

<ResourceLinks repo="https://github.com/communityox/ox_fuel" />

LegacyFuel の代替、またはガソリン缶をアイテムとして使用するためのショーケースとして機能することを目的としたシンプルな燃料リソースです。

## インストール

<Steps>
  ### すべてのリソース依存関係をインストールします。

- [ox_lib](../ox_lib/)
- [ox_inventory](../ox_inventory/)

### ダウンロード[リリース](https://github.com/communityox/ox_fuel/releases)またはソースコードを複製します。

```bash
git clone https://github.com/communityox/ox_fuel.git
```

### オプションの依存関係をインストールします。

これらのリソースは必須ではありませんが、追加機能を提供します。

- [ox_target](../ox_target/)

</Steps>
