# 準備する

Prepare は、頻繁に呼び出されるクエリをより速く実行するために使用され、単一のクエリで使用する複数のパラメータセットを受け入れます。

- Date は、FiveM で一般的に使用される datestring を返しません。
- TINYINT 1 と BIT はブール値を返さない
- 値のプレースホルダのみ使用可能で、列のプレースホルダや名前付きプレースホルダはエラーになります。

とは異なり[rawExecute](./rawExecute)SELECT 文は、選択された列と行の数に応じて、列、行、または行の配列を返す。
