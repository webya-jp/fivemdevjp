# 機能

## 一番近いドア

最も近いドアの鍵を開けようとする。サーバー側のチェックに依存するため、失敗する可能性がある。

```lua
exports.ox_doorlock:pickClosestDoor()
```

## 最も近いドア

最も近いドアと対話する。サーバー側のチェックに依存するため、失敗する可能性がある。

```lua
exports.ox_doorlock:useClosestDoor()
```

## 一番近いドア

一番近いドアを返す。

```lua
exports.ox_doorlock:getClosestDoor()
```

戻る：

* ドア`number`
