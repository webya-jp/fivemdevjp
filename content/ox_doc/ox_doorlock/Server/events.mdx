# イベント

## ハンドラー

これらのイベント**べきではない**他のスクリプトによってトリガーされる。

### ox\_doorlock:stateChanged

ドアの状態が更新されたときにトリガーされる。

```lua
AddEventHandler('ox_doorlock:stateChanged', function(source, doorId, state, usedItem) end)
```

* ソース`number`または`nil`
* doorId：`number`
* の状態だ：`boolean`
* usedItem：`string`または`false`または`nil`
