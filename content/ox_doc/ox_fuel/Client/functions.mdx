# 機能

## セットマネーチェック

内蔵のマネー・チェックを無効にする。

```lua
exports.ox_fuel:setMoneyCheck(method)
```

### パラメータ

* メソッドを使用する：`function(): number`

### 例

```lua
exports.ox_fuel:setMoneyCheck(function()
    local accounts = ESX.GetPlayerData().accounts

    for i = 1, #accounts do
        if accounts[i].name == 'bank' then
            return accounts[i].money
        end
    end

    return 0
end)
```
