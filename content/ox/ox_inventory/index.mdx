# Ox Inventory

An slot-based inventory with item metadata for "item uniqueness".

スロットベースのインベントリシステムで、アイテムの一意性を提供するメタデータ機能を持っています。

## 主な機能

- **スロットベース**: アイテムの配置をスロット単位で管理
- **メタデータサポート**: アイテムごとに固有の情報を保存
- **マルチフレームワーク対応**: ESX、QBCore等に対応
- **高度なUI**: モダンで使いやすいインターface
