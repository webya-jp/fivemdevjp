# 👔 qbマネジメント

## 導入

- 従業員をオンラインまたはオフラインで管理しましょう!
- ボスレベルの個人用隠し場所にアクセスしましょう！
- 複数のボスメニューの場所を許可します。

## 構成

> [!NOTE]
> これらの場所は、`PlayerData.job.isboss`qb-core/shared/jobs.lua ファイルからロールを取得します。

Config.BossMenus = {
    police = {
        vector3(447.16, -974.31, 30.47),
