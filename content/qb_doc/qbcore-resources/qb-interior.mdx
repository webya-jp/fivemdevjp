# 🏠 qb-インテリア

## 導入

- 他のクライアント側ファイルで呼び出せる関数をエクスポートすることで、シェルモデルを生成するためのすべてのロジックを処理します。

## 構成

> [!NOTE]
> エクスポートを追加する場合を除き、このリソースは設定不要です。

## 何が含まれていますか?

> [!NOTE]
> 以下のPDFファイルは、デフォルトで付属するシェルモデルを示しています。

> [!NOTE]
> オプションとして、このリソースはすべての設定済みです。[K4MB1](https://www.k4mb1maps.com/)貝殻！

## 使用例

```lua
RegisterCommand('spawnshell', function()
    local ped = PlayerPedId()
    local coords = GetEntityCoords(ped)
    local shell = exports['qb-interior']:CreateApartmentShell(coords)
end)
```
