# ↖️ qbメニュー

## 導入

その**QBメニュー**システムは、Webベースのユーザーインターフェースと統合されたLuaベースのインタラクティブメニューです。

> [!NOTE]
> メニューアイコンは[フォントオーサム](https://fontawesome.com/) Webサイト

### 特徴

- 動的なメニュー作成
- アイコンと画像のサポート
- メニューの並べ替え機能
- クライアント・サーバー通信
- 使いやすいLuaとJavaScriptインターフェース
- 広範な統合のためのエクスポートとイベント

## メニューを開く

提供されたデータでメニューを開きます

- データ： `table`&#x20;
  - ヘッダー:`string`（オプション）
  - TXT： `string`（オプション）
  - アイコン:`string`（オプション）
  - メニューヘッダー:`boolean`（オプション）
  - 無効： `boolean`（オプション）
  - 隠れた： `boolean`（オプション）
  - アクション： `function`（オプション）
  - パラメータ:`table`（オプション）
    - イベント： `string`
    - 引数:`any`（オプション）
    - サーバー:`boolean`（オプション）
    - isコマンド:`boolean`（オプション）
    - isQBコマンド:`boolean`（オプション）
    - アクション:`boolean`（オプション）
- 選別： `boolean`
- スキップファースト:`boolean`

```lua
exports['qb-menu']:openMenu({
    {
        header = "First Item",
        icon = "icon-name",
        txt = "Description",
        isMenuHeader = false,
        disabled = false,
        hidden = false,
        params = {
            event = "event:name",
            args = { arg1 = "value1" },
            isServer = false,
            isCommand = false,
            isQBCommand = false,
            isAction = false
        }
    },
    {
        header = "Second Item",
        txt = "Another description",
        isMenuHeader = false,
        disabled = true,
        hidden = false
    },
    {
        header = "Third Item",
        txt = "Another description",
        action = function()
           print('You selected the Third Item')
        end
    }

}, true, false)
```

## メニューを閉じる

メニューを閉じる

```lua
exports['qb-menu']:closeMenu()
```
