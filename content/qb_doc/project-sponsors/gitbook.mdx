# 📚 ギットブック

![](/.gitbook/assets/gitbook_logo.png)

## 彼らは誰なの？

あなたが今ご覧になっているウェブサイトは、GitBookのものです！GitBookはドキュメントウェブサイトを専門とする企業です。開発者がGitリポジトリからコンテンツを素早く公開できるように設計されたシンプルなオープンソースツールとしてスタートしました。現在では、技術チームを中心に、ナレッジマネジメントの未来を築く、完全に分散化されたチームへと成長しています。

## 特徴

### コーディング不要

コーディングやデザインを必要とせず、強力なエディターを使用してチーム内の誰もがドキュメントを共同作業できるようになります。

### GitHub統合

ドキュメントは既にデザインされているので、数秒でどこにでも公開できます。GitHub で作業している場合は、Markdown ファイルを同期して、クリックするだけで美しいドキュメントに変換できます。

### チームを作成する

知識を一元管理し、チームと連携して単一の整理されたワークスペースを構築することで、効率性が向上します。全員が同じ情報を共有し、必要な情報を適切なタイミングで見つけることができます。
